## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm

### Setup

1. Clona el repositorio:
```bash
git clone https://github.com/JohannTF/OneTic.git
cd OneTic
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`
